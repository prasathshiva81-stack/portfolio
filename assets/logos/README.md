## After Adding Logos

Once you've downloaded and saved the logos, update the image sources in `index.html`:

Replace:
```html
<img src="https://via.placeholder.com/120/..." alt="..." class="skill-logo">
```

With:
```html
<img src="assets/logos/filename.png" alt="..." class="skill-logo">
```

## Recommended Logo Specifications
- Format: PNG with transparent background
- Size: 512x512px or similar square dimensions
- File size: Keep under 50KB per logo for optimal loading
